방산 SW 엔지니어를 향한 실무 역량 강화 및 자료구조 학습 기록

본 저장소는 하드웨어 제약 조건이 엄격한 방산 임베디드 환경에서 필수적인 C언어 기반 자료구조와 알고리즘을 학습하고 기록하는 공간입니다.

-- 학습 목표 --
메모리 최적화: malloc 사용을 지양하고 정적 배열 및 힙 관리 능력을 배양합니다.

실시간성 확보: 지연 시간(Latency)을 최소화하는 효율적인 데이터 처리 로직을 구현합니다.

코드 신뢰성: 예외 처리(Boundary Check)를 철저히 하여 시스템 중단을 방지하는 코드를 작성합니다.
